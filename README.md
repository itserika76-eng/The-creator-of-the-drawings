# The creator of the drawings

Локальный прототип системы для подготовки чертежей под **КОМПАС-3D**:

- перенос чертежа с изображения в структурированное задание;
- выбор конкретной установленной версии КОМПАС-3D через диалог выбора ярлыка/`exe`;
- генерация чертежа по текстовому запросу с опорой на интернет-источники;
- автоматическое формирование заготовки спецификации;
- валидация по ГОСТ/межотраслевым правилам через настраиваемые чек-листы.

> ⚠️ В этом репозитории реализован **MVP-скелет**. Он не содержит нелицензионных баз ГОСТ и не «загружает всю документацию автоматически». Вместо этого используется локальный реестр стандартов и механизм подключения легальных источников.

## Быстрый старт

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app/main.py
```

## Что уже работает

1. GUI на `tkinter`.
2. Выбор исполняемого файла КОМПАС-3D.
3. Режимы:
   - "По изображению" — анализ файла и генерация JSON-пакета проекта.
   - "По запросу" — сбор фактов из сети (Wikipedia API как заглушка), генерация пакета.
4. Автосоздание:
   - `drawing_package.json` — структурированный черновик чертежа;
   - `specification.csv` — заготовка спецификации;
   - `kompas_macro_template.py` — шаблон макроса для импорта в КОМПАС.
